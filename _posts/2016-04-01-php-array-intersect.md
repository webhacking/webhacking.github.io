배열의 교집합을 계산, `array_intersect` 를 이용

array array_intersect ( array $array1 , array $array2 [, array $ ... ] )

  

    array array_intersect ( array $array1 , array $array2 [, array $ ... ] )

  

array_intersect()는 모든 인수에 존재하는 array1의 모든 값을 포함하는 배열을 반환한다. 키는 보존된다는 것에 주의할것.
